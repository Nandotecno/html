    Proposta de Estrutura para Agendamentos

🌐 Criar uma criar uma aplicação de agendamento para consultórios médicos. 🌐

  • A Unidade deve disponibilizar 4 Salas para escolha.
  • A plataforma deve registrar a escolha do usuário para um determinado dia e horário.
  • Não permitir a escolha do mesmo consultório e horário para mais de 01 usuário.
  • A plataforma deve colher as informações de quem solicitou o agendamento.

    • 🎬 Itens de Cadastro 🎬

  • Nome Completo
  • Telefone
  • E-Mail

    • 🃏 Criar uma tela inicial de login e senha para acesso autorizado para o registro. 🃏
